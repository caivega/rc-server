# rc-server

development server for react component

## Usage

package.json

```js
{
    "devDependencies": {
        "rc-server": "^1.0.0"
    },
    "config": {
        "port": 8001
    },
    "scripts": {
        "start": "node --harmony node_modules/.bin/rc-server",
    }
}
```

```
npm install
npm start
```

open http://localhost:8001/
